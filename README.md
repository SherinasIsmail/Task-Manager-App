# Full-Stack Task Manager

A MERN-stack inspired application developed as part of my technical assignment. This project manages tasks using a React frontend and a Node.js/Express backend.

## 🚀 Features
- **Full CRUD**: Add and Delete tasks with real-time updates.
- **Task Status**: Toggle completion status using checkboxes.
- **Filtering**: Filter tasks by 'All', 'Pending', or 'Completed' status (Bonus Feature).
- **Responsive Design**: Styled with Tailwind CSS for a modern, mobile-friendly look.
- **State Management**: Implements loading and error states for a better user experience.

## 🛠️ Tech Stack
- **Frontend**: React.js, Tailwind CSS (via CDN), Vite
- **Backend**: Node.js, Express.js
- **API**: RESTful architecture

## ⚙️ How to Run Locally

### 1. Backend
- Navigate to the `backend` folder.
- Run `npm install` to install dependencies.
- Start the server using `node server.js`.
- The server will run on `http://localhost:5000`.

### 2. Frontend
- Navigate to the `frontend` folder.
- Run `npm install`.
- Start the development server using `npm run dev`.
- Open the application in your browser.
